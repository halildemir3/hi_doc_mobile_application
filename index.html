colorX():

This method allows the colors to be used as hex code, there is no need to add "0xFF" to the beginning of the colors, just call it easily and give your hex code.

colors.dart:

The colors used in the application are shown here as variables, the desired color can be changed from this file

path.dart:

The file is the file where links and paths are stored in the application. All file paths located under the "assets" folder are located in this file. at the same time, all the icons used in the application are also located in the assets folder in swg format and their paths are defined in this file.

styles.dart:

The styling process has been performed in this file all styling options are defined in this file as variables TextStyles paddings.


Widgets representing screens in the file directory are located in the "views" folder some special components are located under the "widgets"folder. no status management has been used in the application the desired method can be used, and you should create your own route and status management by simply using getX for route management.
